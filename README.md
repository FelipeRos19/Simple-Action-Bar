# Simple-Action-Bar
Classe de Criação de Action Bar

Métodos:

(Para enviar uma mensagem para o Jogador)
ActionBar (nome) = new ActionBar("Texto da ActionBar");
(nome).sendToPlayer(p);

(Para enviar uma mensagem para o Servidor)
ActionBar (nome) = new ActionBar("Texto da ActionBar");
(nome).sendToAll();
